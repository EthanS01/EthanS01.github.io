For the following:


import java.util.Scanner;


public class ConditionalsQuestion {

  public static void main (String [] args) {

    Scanner scnr = new Scanner(System.in);

    int userAge;

    double ticketRate;


    userAge = scnr.nextInt();

    if (userAge < 18) {

       ticketRate = 13.50;

    }

    else if (userAge > 65) {

       ticketRate = 14.50;

    }

    else {

       ticketRate = 16.50;

    }

  }

}


If user input is ‘18’, what value will be output?

A: Error

B: $13.50

C: $14.50

D: $16.50
